FAQ

Is it really free?

Yes, 100% free with no hidden costs, no premium tier, and no ads. The code is open source on GitHub.

Do I need to create an account?

No. There's no signup, no login, no email required. Just open and start using it immediately.

Where is my data stored?

All your data is stored locally in your browser's localStorage. Nothing is sent to any server. If you clear your browser data, your planner data will be reset.

What is the Claude AI tutor?

Claude is an AI assistant by Anthropic. You'll need your own API key from console.anthropic.com to use it. The AI helps with homework, explains concepts, quizzes you, and gives study tips.

Should I use the web version or desktop app?

Both have the same features. The web version works instantly in your browser. The desktop version runs as a native window if you prefer that. Most students use the web version.

Can I use it on my phone?

The web version works on mobile browsers, though it's designed primarily for desktop/laptop screens. The layout adapts to smaller screens.
