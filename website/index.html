How to Install (takes 2 minutes)

1
Click the download button

A ZIP file will download to your computer

2
Unzip the folder

Right-click the ZIP → "Extract All" → Click "Extract"

3
Install Node.js (if you don't have it)

Go to nodejs.org and install the LTS version

4
Open the folder & run the app

Open the folder, double-click the address bar, type cmd and press Enter. Then type:

npm install && npm start

That's it! The app will open automatically. Next time, just run npm start to reopen it.
