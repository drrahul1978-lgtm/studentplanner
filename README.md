# Student Planner

The ultimate school toolkit for students.

**Homework tracker, grade calculator, GPA tracker, study timer, flashcards, notes, class schedule, exam countdown, and AI tutor — all in one free app.**

---

## How to Download & Run

1. Click the green **"Code"** button above, then click **"Download ZIP"**
2. Unzip the folder (right-click → Extract All)
3. Open the `app` folder
4. Double-click **`Install & Run`**

That's it! The app opens automatically.

> **Next time:** Just double-click **`Start`** to reopen the app.

---

## What's Inside

| Feature | Description |
|---------|-------------|
| Homework Tracker | Due dates, priorities, subjects |
| Grade Tracker | Tests, quizzes, projects — see your letter grade |
| GPA Calculator | Auto-calculates your GPA |
| Class Schedule | Weekly view with rooms and times |
| Study Timer | Pomodoro timer with session tracking |
| Flashcards | Create decks, flip, shuffle, study |
| Notes | Full editor with auto-save |
| Exam Countdown | Color-coded urgency for upcoming exams |
| To-Do List | Tasks by category (school, personal, etc.) |
| AI Tutor | Claude AI built in for homework help |
| Study Tools | Quick links to Quizlet, Khan Academy, Desmos, and more |
